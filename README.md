# swamp-slack-blocks

[swamp](https://swamp.club) extension `@mgreten/slack-blocks` — post Block Kit
messages and file attachments to Slack from swamp models and workflows.

📖 **Extension documentation:** [extensions/models/README.md](extensions/models/README.md)

📦 **Install:** `swamp extension pull @mgreten/slack-blocks`

## Repository layout

This repository is a swamp workspace (its own `.swamp.yaml` repo). The
publishable extension lives under [extensions/models/](extensions/models/).
